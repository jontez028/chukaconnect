🔥 One-Time Setup: Connect Your Firebase Database

1Go to console.firebase.google.com → Click "Add project" → Name it chukaconnect → Create project
2Inside your project → Click "Web" (</>) icon → Register app → Copy the firebaseConfig object shown
3Open your index.html file → Find const firebaseConfig = { near the top → Replace the placeholder values with your copied config
4In Firebase Console → Authentication → Get Started → Enable Email/Password sign-in method
5In Firebase Console → Firestore Database → Create database → Start in test mode → Choose a region close to Kenya (e.g. europe-west1)
6In Firebase Console → Storage → Get Started → Start in test mode → This allows image uploads
7Save your updated index.html → Upload to GitHub → Your site now has a real database!
